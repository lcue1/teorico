50 Términos Fundamentales de Desarrollo Web

# Término Definición
1 HTML Lenguaje de marcado estándar para crear páginas web, define la estructura y contenido.
2 CSS Lenguaje de hojas de estilo que describe cómo se presentan los elementos HTML.
3 JavaScript Lenguaje de programación que permite crear contenido dinámico e interactivo en páginas web.
4 DOM Modelo de Objetos del Documento, representación en árbol de la estructura de un documento HTML.
5 API Interfaz de Programación de Aplicaciones, conjunto de definiciones y protocolos para integrar software.
6 HTTP Protocolo de transferencia de hipertexto, base de la comunicación de datos en la web.
7 HTTPS Versión segura de HTTP que utiliza cifrado SSL/TLS para proteger los datos.
8 GET Método HTTP para solicitar datos de un recurso especificado (sin modificar datos).
9 POST Método HTTP para enviar datos a un servidor para crear/actualizar un recurso.
10 isset Función en PHP que verifica si una variable está definida y no es null.
11 Servidor Computadora o sistema que proporciona recursos, datos o servicios a otras computadoras (clientes).
12 Host Computadora o servicio que aloja sitios web o aplicaciones accesibles a través de internet.
13 Dominio Nombre único que identifica un sitio web en internet (ej: ejemplo.com).
14 DNS Sistema de Nombres de Dominio, traduce nombres de dominio a direcciones IP.
15 IP Protocolo de Internet, dirección numérica única asignada a cada dispositivo en una red.
16 Responsive Design Enfoque de diseño que hace que las páginas web se adapten a diferentes tamaños de pantalla.
17 Frontend Parte de una aplicación web con la que interactúa el usuario (cliente).
18 Backend Parte de una aplicación web que se ejecuta en el servidor (lógica, base de datos, etc.).
19 Framework Estructura de software que proporciona funcionalidad genérica para facilitar el desarrollo.
20 Librería Colección de funciones y código preescrito para realizar tareas específicas.
21 CMS Sistema de Gestión de Contenidos, software para crear y administrar contenido digital.
22 SEO Optimización para Motores de Búsqueda, técnicas para mejorar la visibilidad en buscadores.
23 Cookie Pequeño fragmento de datos almacenado en el navegador del usuario por un sitio web.
24 Session Mecanismo para almacenar información del usuario en el servidor durante su visita.
25 AJAX Técnica para crear aplicaciones web interactivas que actualizan contenido sin recargar la página.
26 JSON Formato ligero de intercambio de datos basado en texto, fácil de leer y escribir.
27 XML Lenguaje de marcado extensible para almacenar y transportar datos.
28 REST Estilo arquitectónico para diseñar servicios web escalables y flexibles.
29 CRUD Acrónimo para Create, Read, Update, Delete, operaciones básicas de persistencia de datos.
30 SQL Lenguaje de consulta estructurado para gestionar bases de datos relacionales.
31 NoSQL Tipo de base de datos que no usa el modelo relacional tradicional.
32 MVC Patrón de diseño Modelo-Vista-Controlador para separar lógica, interfaz y control.
33 Git Sistema de control de versiones distribuido para rastrear cambios en el código.
34 CDN Red de Distribución de Contenido, sistema de servidores que entrega contenido rápidamente.
35 UI Interfaz de Usuario, medios por los cuales un usuario interactúa con un sistema.
36 UX Experiencia de Usuario, percepción y respuesta de un usuario al usar un producto/servicio.
37 SSL/TLS Protocolos criptográficos que proporcionan seguridad en las comunicaciones.
38 Cache Almacenamiento temporal de datos para acelerar futuras solicitudes de esos datos.
39 WebSocket Protocolo de comunicación que permite conexiones bidireccionales persistentes.
40 JWT JSON Web Token, estándar para crear tokens de acceso que permiten la autenticación.
41 OAuth Protocolo estándar para autorización que permite acceso seguro sin compartir credenciales.
42 Middleware Software que conecta componentes o aplicaciones, procesando datos entre ellos.
43 Virtual DOM Concepto donde se mantiene una representación ideal del DOM en memoria y se sincroniza.
44 Polyfill Código que implementa funcionalidades en navegadores que no las soportan nativamente.
45 Webpack Empaquetador de módulos para aplicaciones JavaScript modernas.
46 Babel Transpilador de JavaScript que convierte código moderno en versiones compatibles.
47 CI/CD Integración Continua/Despliegue Continuo, prácticas para automatizar el desarrollo.
48 SPA Aplicación de Página Única, aplicación web que carga una sola página y la actualiza dinámicamente.
49 PWA Aplicación Web Progresiva, aplicación web que usa tecnologías modernas para ofrecer experiencia similar a apps nativas.
50 Web Components Conjunto de tecnologías que permiten crear elementos personalizados reutilizables.
